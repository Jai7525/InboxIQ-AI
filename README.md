# InboxIQ AI

AI-powered inbox intelligence dashboard with FastAPI backend and React frontend.

## Project Structure

- `backend/` - FastAPI API, Gmail sync, RAG chat, search, analytics, and threat detection.
- `frontend/` - React/Vite dashboard UI.

## Development

Backend:

```bash
uvicorn backend.main:app --reload
```

Frontend:

```bash
cd frontend
npm run dev
```
