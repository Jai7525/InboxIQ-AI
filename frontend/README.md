# InboxIQ AI Frontend

React + Vite + Tailwind CSS dashboard for InboxIQ AI, a RAG-based semantic inbox intelligence and threat detection system.

## Scripts

```bash
npm install
npm run dev
npm run build
```

## Structure

- `src/components` contains reusable layout, module, and UI components.
- `src/pages` contains product pages with page-level mock data.
- `src/context/ThemeContext.jsx` manages light and dark mode with the root `dark` class.
- `src/services/api.js` is a placeholder API service layer for future backend integration.
